# Amazon Canada Manual Testing

## Objective

To verify that key features of the Amazon Canada website function correctly and provide a positive user experience.

## Scope

The following features will be tested:

- Search functionality
- Product navigation
- Shopping cart
- Sign-in page validation

## Testing Method

Manual Functional Testing

## Test Environment

Operating System: Windows 11

Browser: Google Chrome

## Entry Criteria

- Website is accessible
- Browser is functioning correctly
- Internet connection is available

## Exit Criteria

- All planned test cases have been executed
- Results have been documented
- Any defects identified have been recorded

## Deliverables

- Test Plan
- Test Cases
- Bug Reports
- Screenshots
- Test Execution Results
